Go
==========================================
A Go program basically consists of the following parts:
- Package Declaration.
- Import Packages.
- Functions.
- Variables.
- Statements & Expressions.
- Comments.
===========================================
Go do not used character @, $, % within identifiers.
ex: tile, _tile, t_123ile (true)
123@sd, 1a, ee@aa, e% (false)
